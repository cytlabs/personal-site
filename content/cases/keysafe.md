# KeySafe 去中心化密码管理工具

场景：全栈产品 / Web3 工具 / 数据安全 / AI 辅助开发

## 背景

Web3 用户对数据主权、隐私保护、去中心化存储和钱包身份有更强需求，因此尝试设计一个融合 Web2 密码管理体验和 Web3 数据主权理念的密码管理工具。

## 问题

- 密码数据属于高敏感信息，需要端到端加密。
- 钱包登录不能直接替代主密码，需要额外安全因子。
- IPFS 适合存储加密数据，但需要处理索引、同步状态和 CID 管理。
- Web3 产品门槛高，需要尽量保留 Web2 使用体验。

## 行动

- 使用 Next.js、React、TypeScript、Ant Design 构建前端。
- 使用 PostgreSQL + Prisma 存储非敏感索引数据。
- 集成 MetaMask 钱包登录，通过钱包签名参与密钥派生。
- 使用 PBKDF2 和 AES-256-GCM 实现本地加密。
- 使用 IPFS / Pinata 存储加密后的密码数据。

## 结果

- 完成可演示的 Web3 密码管理工具原型。
- 支持钱包登录、主密码验证、密码增删改查、加密存储、IPFS 同步和 CID 查询。
