# 案例摘要

## 达人营销 AI 工作流

把客户 Brief、项目启动、达人匹配、外联话术和异常提醒拆成可执行 AI Workflow。重点价值是把自然语言需求转成结构化流程，并明确规则、LLM、表格和人工确认的边界。

## KeySafe 去中心化密码管理工具

Next.js + PostgreSQL / Prisma + 钱包登录 + AES-256-GCM + IPFS 的全栈原型。重点价值是证明能独立完成产品设计、工程实现、安全边界和演示闭环。

## Job Search Materials Skill

把求职资料、技能矩阵、项目故事、简历版本和面试话术生成流程标准化。重点价值是把一次性资料整理变成可迭代的个人工作流。

## GitOps 自动化发布平台

引入 GitOps，将基础设施和应用配置统一放入 Git 管理，搭建 GitLab Runner + Kustomize + ArgoCD 自动化发布平台。重点价值是把发布流程标准化、可审计、可回滚。
